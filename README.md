## ✨ What's Inside

- 💘 **Valentine's Ask Page** — animated YES/NO interaction where the NO button runs away
- 🎉 **Celebration Page** — fireworks, floating kisses, background music on YES
- 💌 **Love Letter** — a personalized, glowing letter
- 📸 **Photo Heart Grid** — 27 photos arranged in a perfect heart shape with click-to-zoom
- 🎵 **Background Music** — auto-plays on interaction
- 💫 **Full Animations** — floating hearts, sparkles, glowing text, gradient backgrounds

## 🛠️ How to Customize It for Yourself

### 📸 Step 1 — Add Your Photos

- Create a folder named `images/` in the project root
- Add exactly **27 photos** named `1.jpg`, `2.jpg` ... `27.jpg`
- Use **square or near-square** `.jpg` images for the best heart-grid look
- Keep each photo under **500KB** for fast loading
- 💡 Need free photos? Try [Unsplash](https://unsplash.com) or [Pexels](https://pexels.com)

### 🎵 Step 2 — Add Your Song

- Place your song file in the **root folder** (same level as `index.html`)
- Rename it to exactly `love.mp3`
- Must be `.mp3` format
- 💡 Need to convert? Use [CloudConvert](https://cloudconvert.com) — free & easy

### 💌 Step 3 — Edit the Letter

- Open `index.html` in any text editor (Notepad, VS Code)
- Find the `<div class="letter">` section
- Replace the letter content with your own words 💕

### 🔤 Step 4 — Change the Name

- Search for `Mr.Rohit` in `index.html`
- Replace with your own name
